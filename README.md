# 🛡️ Sovereign Ledger Alghwairy - v1.0.0
## منظومة الميزان السيادي للمؤسسات المالية

Welcome to the **Sovereign Ledger Alghwairy**, a high-security, professional financial accounting system designed for the year 2026. This application is built with React, TypeScript, and Electron, focusing on offline-first reliability and ZATCA compliance.

---

## 🛠️ Quick Start

### 1. Installation
Ensure you have [Node.js](https://nodejs.org/) installed.
```bash
npm install
```

### 2. Development Mode
Run the application in development mode:
```bash
npm run dev
```
Then, in a separate terminal:
```bash
npm run electron
```

### 3. Build & Package (Create Installer)
To generate the Windows installer (`.exe`):
```bash
npm run dist
```
The installer will be generated in the `release/` directory.

## 🔐 Security Protocols
- **Lockdown Logic:** Automated system lock after 3 failed login attempts.
- **Biometric Ready:** Integrated fingerprint authentication support.
- **Sovereign Encryption:** Local database encryption for maximum privacy.

## ⚖️ Compliance
- **ZATCA Phase 2:** Fully compatible with Saudi Arabian e-invoicing standards.
- **Audit Trails:** Every transaction is logged and archived in the sovereign trash bin.
